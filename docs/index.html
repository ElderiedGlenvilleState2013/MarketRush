Market Rush 소개

기존의 e-Commerce 어플이 기본적으로 가지고 있는 주요 기능들이 포함되어 있는 심플한 온라인 식료품 마켓 어플 입니다. 네이버 쇼핑 api를 통해 사용자가 원하는 식료품들을 빠르고 쉽게 찾아 나만의 장바구니에 담기가 가능한 어플입니다.

Key Features

  • 간편분류를 통한 상품보기

    직관적인 간편 분류에 따라 Naver에서 제공하는 검색 API를 바탕으로 상품조회가 가능합니다.

  • 장바구니 바로 담기 기능

    세부 카테고리에서는 스크롤을 내리면서 바로 사용자가 원하는 상품을 장바구니에 담을 수 있습니다.

  • 간편한 장바구니 관리

    Realm DB를 이용한 간편한 장바구니 관리가 가능합니다.
